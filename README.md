# ⚡ NERV MAGI SYSTEM v8.0 — Terminal User Interface (TUI)

🚀 **Overview**  
NERV MAGI TUI is a fully interactive Terminal User Interface (TUI) that recreates the iconic NERV supercomputer system from *Neon Genesis Evangelion*. Built in pure Python with `curses`, it transforms your Linux terminal into a functional command center where you can control Evangelion units, engage angels in tactical combat, customize weapon loadouts, and experience the atmosphere of NERV headquarters.

---

## ✨ Features

### Core Systems
- **MAGI Supercomputer**: Three voting units (CASPAR / MELCHIOR / BALTHASAR) with live voting and consensus calculation  
- **EVA Status Monitoring**: Real-time sync rates, power levels, AT Field strength for EVA-00/01/02 with live fluctuation  
- **Tactical Battlefield**: 30×18 grid map of Tokyo-3 with terrain, defense lines, and angel AI  
- **Turn-Based RPG Combat**: Battle unique angels with abilities, party switching, and pilot stats  
- **Weapon Loadout Editor**: 10 weapons with damage, accuracy, range, and special effects  
- **Deploy System**: Choose which EVAs to send into battle  
- **Angel Radar**: Live tracking with sweep arm, bearing/range, HP bar, and abilities  
- **Command Terminal**: Type commands to control all systems with fuzzy matching  

### Interactive Features
- **3 Minigames**:  
  - *AT Field Breach*: Shooter  
  - *Sync Calibration*: Rhythm game  
  - *NERV Trivia*: Quiz  
- **SEELE Mode**: Toggle ominous 12-monolith overlay  
- **Hospital Easter Egg**: ASCII video player with 64 scenes  
- **Pilot Dialogue**: Characters speak during battle with unique personalities  
- **Alert System**: Scrolling NERV-style alerts with priority levels  
- **Berserk Mode**: EVA-01 can go berserk at high sync rates  

---

## 📋 Requirements

- **Operating System**: Linux, macOS, or Termux (Android)  
- **Python**: 3.6+  
- **Terminal**: 256-color support (most modern terminals)  
- **Minimum Size**: 120×36 characters for full experience
